# 회기·석계 술집 가이드 — 설계 스펙 (2026-07-16, 사용자 승인됨)

## 목적
japan-food-guide(일본 5도시 350곳)와 같은 형식의 국내판 술집 가이드.
지역: 회기역 + 석계역, 지역당 20곳(총 40곳).

## 확정 사항 (브레인스토밍 Q&A)
- **구조**: 새 독립 사이트. japan-food-guide 디자인·필터·카드 구조 재사용.
- **평점 소스**: 네이버 플레이스 단독 — 방문자리뷰 수 + 블로그리뷰 수(별점 없음), 네이버플레이스 딥링크.
- **카테고리**: 고정 세트 없음. 리서치 결과의 실제 지역 분포로 확정. **바·와인바 제외.**
- **배포**: 로컬 HTML + git 로컬 커밋만. 공개(GitHub/Pages)는 별도 지시 시.

  area: '회기' | '석계',

japan판의 산토리 tier / 타베로그 / 미슐랭 / STATION_MAP 번역 로직은 전부 삭제(국내 데이터라 불필요).

## 선별 기준
- 네이버 방문자리뷰 수 상위 + 블로그 언급 교차 검증
- 폐업 여부 확인 필수
- 바·와인바 제외. 술 파는 음식점(고기·횟집·전·포차·이자카야·호프 등) 중심
- 회기(경희대 대학가)와 석계(로컬 먹자골목) 지역 성격 차이를 분포에 반영

## UI
- japan판 최신 디자인 그대로: Zen Kaku Gothic New 폰트, 다크 기본+토글,
  매거진 리더카드 + 그리드 카드(1a), 언더라인 탭 지역내비(4c), 모바일 2열 타일 펼침
- 도시 탭 자리에 회기/석계 탭
- 타베로그 배지 자리 → 네이버 리뷰수 표기(방문자리뷰 N · 블로그 M)

## 검증
- `node --check` 문법, DATA.length=40, 지역당 20, 중복명·중복사진 0, photo HTTP 200
- Playwright: 데스크톱/모바일 스크린샷 + 탭·필터·테마토글 클릭 검증
